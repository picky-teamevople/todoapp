# Design Token 최종 기준표

> **비교 대상**: [CONTEXT.md](../CONTEXT.md) (Notion PRD v1.1 + 디자인 스펙 v2.1) vs Figma 파일 `구글스티치 강의` (fileKey `4pBEoZtyuf93YE9Rm1N14S`) 노드 `5:64`("캘린더 뷰") / `5:266`("할 일 상세 보기" 모달) / `5:398`("설정 페이지") / `5:558`("메인 대시보드").
> **참고**: 4개 Figma 노드는 사이드바 내비게이션·캘린더·서브태스크·코멘트·우선순위·설정·대시보드 등 Notion PRD의 단순 To-Do 리스트보다 넓은 스코프의 화면이라, 일부 역할(카테고리 태그, 완료율 게이지, 삭제/경고 색상 등)은 화면에 직접 노출되지 않았습니다. 이 경우 "문서 기준 유지"로 표기했습니다. Figma 파일에 연결된 Variables는 없어(`get_variable_defs` 결과 없음) 전부 노드에 하드코딩된 raw 값 기준입니다.
> **채택 규칙**: 피그마 값 우선 채택 / 충돌 시 이유 명시 / 피그마에만 있으면 "신규 발견" / 노션에만 있으면 "문서 기준 유지".

## 신규 발견 항목

(피그마에서만 발견된 값들)

- **사이드바 전용 다크 배경색** `#150502` — Text Primary(`#311B14`)와 별개의 더 짙은 톤. 다크 사이드바 배경 전용 색상으로 별도 토큰화 필요.
- **콘텐츠 표면 크림톤 다단계**: `#FFF8F1`(모달/캘린더 셀 배경), `#FBF2E7`(설명 박스/댓글 입력 배경), `#EFE7DC`(Bento 블록) — Surface(`#EDE6D8`) 외에 3단계 표면톤이 추가로 존재.
- **디스플레이 세리프 폰트**: `Source Serif 4` — Pretendard 단일 서체 정책과 별도로 타이틀/헤딩 전용 세리프 폰트가 도입됨.
- **섹션 레이블 패턴**: 14px SemiBold + uppercase + letter-spacing 0.7px (Hanken Grotesk) — 노션 문서에 없는 별도 컴포넌트 스타일(DESCRIPTION, SUBTASKS, TAGS 등).
- **캘린더 이벤트 인디케이터 색상 2종**: `#8C7365`(브라운), `#D67D5C`(테라코타) — 태그 칩 색상과는 분리된 별도 accent 세트.
- **스페이싱 12px / 48px** — 기존 4/8/16/24/32 스케일에 없던 값. 특히 12px은 사용 빈도가 매우 높음.
- **배지/태그 radius 2px** — 버튼(4px)·카드(8px) 체계와 분리된 별도 radius.
- **카드 그림자 값**: `drop-shadow(0px 4px 6px rgba(49,27,20,0.04))`(사이드 패널/캘린더 그리드), `0px 8px 24px rgba(49,27,20,0.08)`(모달) — 노션 문서엔 "은은한 그림자"로만 기술되어 구체 수치가 없었음.
- **상태 뱃지 / 속성 패널 UI**: "In Progress", "TODAY", "1/3 Completed", STATUS/DUE DATE/PRIORITY/PROJECT/TAGS 속성 카드 — Notion PRD 스코프에 없는 "할 일 상세 모달" 전용 컴포넌트.
